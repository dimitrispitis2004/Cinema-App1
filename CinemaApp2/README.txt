# Cinema App

Μια εφαρμογή κινηματογράφου για κράτηση θέσεων μέσω κινητής συσκευής / web.

## Οδηγίες εγκατάστασης


Πήγαινε στον φάκελο backend:
cd backend

Εγκατάστησε τις εξαρτήσεις:
npm install

Τρέξε τον server:
npx nodemon app.js

Frontend
Πήγαινε στον φάκελο frontend:
cd frontend

Εγκατάστησε τις εξαρτήσεις:
npm install

Τρέξε το frontend:
npm start
