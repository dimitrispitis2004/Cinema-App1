# Cinema App

Μια εφαρμογή κινηματογράφου για κράτηση θέσεων μέσω κινητής συσκευής / web.

## Οδηγίες εγκατάστασης


Πήγαινε στον φάκελο backend:
cd backend

Εγκατάστησε τις εξαρτήσεις:
npm install

Τρέξε τον server:
npx nodemon app.js

Δημιουργία νέου React project :
npx create-react-app frontend

Μεταφορά του υπάρχοντος App.js και App.css στον νέο φάκελο frontend/src/.
Αν υπάρχει παλιό App.js και App.css, κάνε overwrite.

Frontend
Πήγαινε στον φάκελο frontend:
cd frontend

Εγκατάστησε τις εξαρτήσεις:
npm install
npm install axios


Τρέξε το frontend:
npm start

Λειτουργικότητα
Εγγραφή / Σύνδεση χρηστών (JWT authentication)
Προβολή κινηματογράφων
Προβολή ταινιών ανά cinema
Κράτηση θέσεων με επιλογή ημερομηνίας, ώρας και θέσεων
Προβολή ιστορικού κρατήσεων
